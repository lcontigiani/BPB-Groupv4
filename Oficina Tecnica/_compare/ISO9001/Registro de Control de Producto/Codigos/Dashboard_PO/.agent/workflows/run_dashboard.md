---
description: How to start the BPB Control Dashboard
---

1. Open your terminal (CMD or PowerShell).
2. Navigate to the project directory:
   ```bash
   cd "\\bpbsrv03\lcontigiani\Oficina Tecnica\Registro de Control de Producto\Codigos\Dashboard_PO"
   ```
3. Run the application:
   ```bash
   python app.py
   ```
4. Open your browser and go to:
   [http://127.0.0.1:5000](http://127.0.0.1:5000)
