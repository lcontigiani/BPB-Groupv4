# Herramientas Tecnológicas

## Stack Principal

| Herramienta | Categoría | Uso |
|-------------|-----------|-----|
| **Bitrix24** | CRM / Comunicación | Comunicaciones internas, gestión de clientes, tareas y proyectos |
| **Microsoft 365** | Productividad | Email (Outlook), documentos (Word, Excel, PowerPoint), OneDrive |
| **Claude Teams** | IA Corporativa | Asistente de IA para el equipo (en implementación) |

## Formatos de Archivo

| Formato | Uso |
|---------|-----|
| .xlsx | Datos, análisis, tablas |
| .docx | Documentos, reportes, contratos |
| .pptx | Presentaciones |
| .pdf | Documentos formales, archivos |
| .json | Exportaciones de sistemas, datos estructurados |
| .csv | Exportaciones masivas, bases de datos |
