# Repositorios de GitHub Recomendados

Colección curada de los repositorios más útiles para potenciar tu uso de Claude.

---

## Repositorios Esenciales (Tier 1)

### obra/superpowers
**El framework original de "superpowers" para Claude Code**
- 20+ skills battle-tested (TDD, debugging, colaboración)
- Comandos: /brainstorm, /write-plan, /execute-plan
- Skills-search tool integrado
- Metodología de desarrollo de software agentico
- **Link:** https://github.com/obra/superpowers

### Piebald-AI/claude-code-system-prompts
**System prompts completos de Claude Code, actualizados por versión**
- Todos los system prompts de Claude Code desglosados
- 24 descripciones de tools integrados
- Prompts de sub-agentes (Plan/Explore/Task)
- Actualizado a minutos de cada release
- **Link:** https://github.com/Piebald-AI/claude-code-system-prompts

### modelcontextprotocol/servers
**Repositorio oficial de MCP Servers de Anthropic**
- Servers oficiales mantenidos por Anthropic
- filesystem, github, slack, postgres, sqlite, y más
- Referencia para crear tus propios servers
- **Link:** https://github.com/modelcontextprotocol/servers
